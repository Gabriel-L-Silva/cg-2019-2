Todas as funcionalidades solicitadas na descrição da tarefa foram implementadas. Na implementação dos itens A1 e A3, criou-se uma classe Collection que implementa uma lista genérica 
de referências baseada na classe std::vector. Esta classe possui funções de adição, remoção, iterador e uma função que verifica se a coleção de elementos está vazia, necessárias para 
cada coleção de objetos de cena/componentes, porém as mesmas são sobrescritas na classe SceneObject pois possuem detalhes diferentes de implementação.
Para funcionalidades extras foram implementadas combinações de teclas de atalho para criação e deleção de objetos de cena no nó corrente.
