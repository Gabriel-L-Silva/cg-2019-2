Todas as funcionalidades solicitadas na descrição da tarefa foram implementadas e não houveram funcionalidades extras 
implementadas. Na implementação dos itens A1 e A3, criou-se uma classe Collection que implementa uma lista genérica 
de referências baseada na classe std::vector. Esta classe possui funções de adição, remoção e iterador necessárias
para cada coleção de objetos de cena/componentes, porém as mesmas são sobrescritas na classe SceneObject pois possuem
detalhes diferentes de implementação.
